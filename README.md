#  Sistema de Inventario para Tienda Levi's

Este proyecto es un sistema de gestión de inventario desarrollado en **Laravel** y **MariaDB/MySQL**, pensado para una tienda revendedora y distribuidora de productos Levi's. La cual permite controlar productos, proveedores, compras, ventas y herramientas internas.

---

##  Módulos incluidos

- **Productos**: Registro, edición, stock y precios
- **Proveedores**: Gestión de contactos y compras
- **Compras / DetalleCompra**: Registro de compras y desglose por producto
- **Ventas / DetalleVenta**: Flujo de ventas con estados y detalle por producto
- **Clientes**: Registro y seguimiento
- **Herramientas**: Control de uso interno (no para venta)

---

## ⚙️ Requisitos

- PHP 8.x
- Composer
- XAMPP (Apache + MariaDB)
- Visual Studio Code (opcional)
